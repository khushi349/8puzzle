# KIIT University Frontend Portal

A frontend-focused university web portal developed using HTML, CSS, and PHP.  
The project provides an interactive and responsive interface showcasing campus overview, university history, courses offered, fee structure, placement information, and an admin login section.

## Features
- Responsive university website layout
- Fixed navigation menu
- Campus overview section with images
- Courses and fee structure display
- Internship and placement information
- Admin login interface

## Technologies Used
- HTML5
- CSS3
- PHP

## Purpose
This project was developed for academic and learning purposes to practice frontend web development and webpage structuring.
